# Resume Directory

Place your resume PDF file here.

**File name:** `resume.pdf`

The resume will be:
- Linked in the Hero section ("Resume" button)
- Previewed in the Resume section (iframe)
- Available for download

Make sure to:
- Keep it updated with your latest experience
- Use a clean, professional format
- Optimize file size (keep under 2MB for web)
- Use PDF format for universal compatibility

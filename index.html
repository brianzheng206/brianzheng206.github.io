Projects

A selection of my recent work and engineering projects

End-to-End Autonomous Drone

Autonomous drone delivery system with real-time object detection and AI-driven flight control.

C++ Python ROS2 TensorFlow YOLOv4 AirSim PX4 ArduPilot
Problem: Create an autonomous delivery system capable of navigating complex campus environments and accurately delivering payloads.
Approach: Developed custom object detection models using YOLOv4-tiny and integrated them into a real-time drone control pipeline via ROS2 and MAVSDK.
Result: Successfully simulated and deployed flight missions in AirSim and PX4 environments with end-to-end autonomy from navigation to payload release with onboard AI inference.

Real-Time Expression Detector

Personalized facial expression recognition system using deep learning and computer vision.

PyTorch MediaPipe GStreamer OpenGL GRU
Problem: Build a real-time, personalized facial expression recognition system that runs efficiently on standard hardware.
Approach: Built a real-time inference pipeline using MediaPipe Face Mesh for landmarks and a lightweight PyTorch GRU classifier (hidden-64) with class weighting, label smoothing, and gradient clipping.
Result: Achieved 30+ FPS with webcam → landmarks → GRU → EMA smoothing pipeline, with validation macro-F1 reporting and confusion matrix analysis.

6-DOF CV Robotic Arm

Computer vision-controlled robotic arm with gesture tracking and inverse kinematics.

C++ Python MediaPipe OpenCV SOLIDWORKS
Problem: Create a robotic arm controlled by hand gestures and color tracking for intuitive human-robot interaction.
Approach: Implemented MediaPipe and OpenCV's HSV color thresholding to detect and track gestures, extracting positions using image moments. Designed arm using SOLIDWORKS with PID control and Inverse Kinematics logic.
Result: Successfully controlled a 6-DOF robotic arm and gripper through real-time gesture and color tracking.

About Me

I'm a Mechatronics Engineering student at the University of Waterloo, specializing in autonomy software development and perception systems. Currently, I'm working as an Autonomy Software Developer at enVgo, where I optimize AI models and develop real-time computer vision pipelines for autonomous vessels.

At WATonomous, I lead a team of 40+ members developing an autonomous Kia EV, focusing on sensor fusion, 3D object detection, and path planning. My work spans the full autonomy stack—from low-level sensor drivers to high-level AI inference and control algorithms.

I'm passionate about building intelligent systems that bridge the gap between perception, planning, and control. Whether it's optimizing YOLOv8 models with TensorRT, implementing Structure-from-Motion for 3D reconstruction, or developing real-time multi-camera stitching with GStreamer, I love tackling challenging problems in robotics and computer vision.
